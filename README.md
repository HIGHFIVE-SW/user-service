## 📝 프로젝트 소개
Trendist는 세상의 흐름을 읽고, 나에게 맞는 대외활동을 찾는 플랫폼입니다. <br/>
단순한 정보 탐색을 넘어서 자신에게 필요한 글로벌 이슈/대외활동을 제안받고, <br/>
그 결과를 나만의 포트폴리오로 쌓아갈 수 있습니다.

<br />

## 📝 레포지토리 소개
user-service는 사용자 및 티어와 관련한 기능들이 구현되어 있는 레포지토리입니다. <br/>
